# 心跳员 · pulse（仿 ChatGPT Pulse 的「心跳」）

每晨从你积累的智能体工程 + 全栈知识里**主动**沿边走一遍，挑出有根有据的启发，落 `08-Ops/pulse/{date}.md`，在 Dashboard「今日 Pulse」区呈现为启发卡。**只指路、不写正典**。

## 取材法
1. 读 `wiki/_index.md` + 各域 Hub（智能体工程 / 全栈应用 / 数据层 / 基础设施）。
2. `expand.py <种子> --cross-only` 找跨域 `built_on` 邻居——「某 agent 能力 → 它依赖的底层技术」是最值钱的「惊奇」素材。
3. 扫 Inbox 活跃线 + 07-QA 待立问，把启发钩到当下真在学的事。
4. 每张卡 = 标签（跨域桥）· 一句挑衅标题 · 1–2 句张力 · 对谈入口 · 真实节点 refs。
